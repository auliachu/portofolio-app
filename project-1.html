Project Overview

Movies and Musical Online Reservation adalah website pemesanan tiket musikal atau tiket movie secara online. Website dibagi menjadi dua halaman utama yaitu halaman user dan halaman admin. Halaman user berguna untuk user mendaftarkan akun agar dapat melakukan pemesanan tiket movie dan musical secara online, user dapat memilih jenis tiket yang ingin dibeli, user dapat menentukan kursi yang ingin dipilih, user masuk ke halaman keranjang dan dapat menambah pesanan kursi jika ingin lebih dari satu lalu melakukan checkout sehingga kemudian masuk ke halaman nota pembayaran. User melakukan pembayaran dengan memilih jenis pembayaran yang ingin digunakan, setelah melakukan pembayaran maka status pembayaran akan berubah menjadi sudah dibayar. selanjutnya user menunggu konfirmasi dari pihak admin. setelah dikonfirmasi oleh pihak admin maka status berubah menjadi Lunas dan user dapat mencetak tiket.

Halaman Admin berguna untuk menambahkan musikal dan movie, mengatur jadwal tayang, menentukan jenis tiket yang tersedia dan harga pada masing-masing jenis tiket. Admin juga mengatur jika ada diskon tertentu pada masing-masing jenis tiket. Admin juga melakukan konfirmasi pada setiap faktur yang masuk. Lalu admin dapat mengelola laporan penjualan tiket musikal dan movie. Admin juga dapat menghapus musikal dan movie jika sudah habis tayang, admin juga dapat mengupdate status pembayaran user menjadi Lunas atau Gagal sehingga user dapat mencetak atau mendapatkan tiket.
